# Assets para el instalador

Necesitas agregar los siguientes archivos de icono:

## Archivos requeridos:

### Para Windows:
- `icon.ico` - Icono principal (256x256 px, formato ICO)
- `tray-icon.png` - Icono del system tray (16x16 o 32x32 px)

### Para macOS:
- `icon.icns` - Icono principal (formato ICNS con múltiples tamaños)
- `tray-icon.png` - Icono del menu bar (16x16 px)

### General:
- `icon.png` - Icono PNG (512x512 px) para uso general

## Cómo crear los iconos:

### Opción 1: Herramienta online
1. Ve a https://www.icoconverter.com/ para crear .ico
2. Ve a https://cloudconvert.com/png-to-icns para crear .icns

### Opción 2: Desde tu logo existente
Usa el logo de CommandPulse que ya tienes y conviértelo a los formatos requeridos.
